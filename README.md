_[Homebrew]_ `reset`
===================
A simple Home`brew` [external command] to `reset` an [`install`]ed _[Cask]_ according to its [`zap`] [_stanza_], without [`uninstall`]ing the associated app.

Install
-------
~~~ sh
brew tap danielbayley/reset
brew reset #cask
~~~
